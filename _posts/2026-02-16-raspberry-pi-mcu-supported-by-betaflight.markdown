TIL Raspberry Pi launched MCU [RP2350](https://www.raspberrypi.com/documentation/microcontrollers/silicon.html#rp2350) which has 2 Cortex-M33 cores at up to 150 MHz and [RP2040](https://www.raspberrypi.com/documentation/microcontrollers/silicon.html#rp2040) which has 2 Cortex-M0+ cores at 133 MHz and Betaflight supports them [RP2350 Lands in Betaflight!](https://www.betaflight.com/blog/2025/10/10/RP2350%20Lands%20in%20Betaflight). I don't know if 2x150Mhz is better then single core at 450Mhz which some flight controllers have for quite time.
